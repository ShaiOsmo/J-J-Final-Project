# AI Diagnostic Platform for CARTO™ 3

## Overview

Built an AI-powered diagnostic platform using Python and MongoDB to automate complaint investigation and root cause analysis for CARTO™ 3 medical devices.

The platform integrates Llama 3, LangChain, and LlamaIndex for intelligent analysis and knowledge retrieval, reducing investigation time from weeks to minutes while improving diagnostic accuracy and operational efficiency.

## Technologies

- Python
- MongoDB
- Llama 3
- LangChain
- LlamaIndex
- HTML
- JSON

## Key Features

- Automated complaint investigation
- AI-assisted root cause analysis
- Knowledge retrieval using LLMs
- MongoDB data storage
- Structured diagnostic workflow

## Project Structure

- `agent/` – AI agent and RCA logic
- `extractors/` – Data extraction modules
- `utils/` – Utility functions
- `UI/` – User interface
- `data/` – Input data
